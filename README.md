## Webhook separati

Apri `script.js` e cambia:

```js
const APPOINTMENT_WEBHOOK = "INSERISCI_WEBHOOK_VISITE";
const RECRUITMENT_WEBHOOK = "INSERISCI_WEBHOOK_BANDI";
const STAFF_PASSWORD = "medical123";
```

- `APPOINTMENT_WEBHOOK`: canale Discord richieste visite.
- `RECRUITMENT_WEBHOOK`: canale Discord candidature bandi.
